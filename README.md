# AI Customer Support Automation

A university project demonstrating workflow automation using **Make.com**, **OpenAI**, **Google Sheets**, and **AI Agents** to automate repetitive customer support enquiries.

## Overview

This project was developed to address the challenge of managing high volumes of repetitive customer support enquiries. The solution automates the generation of professional customer responses using AI, reducing manual workload and improving response efficiency.

## Technologies Used

- Make.com
- OpenAI
- Google Sheets
- Make AI Agents
- Prompt Engineering

## Key Features

- Automated customer support responses
- AI-powered workflow automation
- Google Sheets integration
- AI Agent implementation
- Prompt engineering
- Workflow testing and validation

## Workflow 1

**Modules**

Google Sheets → OpenAI → Google Sheets

This workflow automatically detects new customer enquiries entered into a Google Sheet, generates a professional AI response using OpenAI, and writes the response back into the spreadsheet.

## Workflow 2

**Modules**

Google Sheets → Make AI Agent → Google Sheets

This workflow uses a Make AI Agent configured with business rules and company policies to generate accurate, context-aware customer support responses.
